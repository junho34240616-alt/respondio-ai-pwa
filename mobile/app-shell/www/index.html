Respondio Mobile Shell

원격 세션 센터를 WebView/iframe으로 감싸는 실제 앱 셸 프로토타입입니다.

Bridge State

현재 로그인 요청

아직 받은 플랫폼 로그인 요청이 없습니다.

Flow
  1. Respondio 세션 센터를 연다
  2. 사용자가 iframe/WebView 안에서 Respondio 로그인
  3. 직접 로그인 준비 후 앱 브리지 요청 수신
  4. 실제 앱에서는 플랫폼 로그인 WebView를 띄운다
  5. 성공/실패/만료 결과를 다시 세션 센터로 보낸다
